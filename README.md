# Blackjack Trainer Pro

Entrenador educativo y simulador de blackjack. **Uso educativo únicamente**: la app no promete
ganancias, no procesa dinero real y no debe utilizarse para asistir juego en casinos.

## Características

- Curso progresivo (reglas, estrategia básica, Hi-Lo, true count, apuestas, desviaciones).
- Simulador de mesa con hit, stand, double, split, surrender, insurance.
- Conteo automático Hi-Lo con running count y true count (redondeo configurable).
- Drills de conteo y de true count con retroalimentación.
- Entrenador de estrategia (hard / soft / pairs / mixto).
- Entrenador de desviaciones (Illustrious 18 + Fab 4 resumidas).
- Configuración completa de reglas y rampa de apuestas simuladas.
- Estadísticas locales con gráficas.
- PWA instalable en móvil.
- Diseño mobile-first y responsive.
